# Student Management System (Python + SQLite)

This is a simple console-based Student Management System built using Python and SQLite.
It allows users to manage student records using basic CRUD operations.

## Features
- Add a new student
- View all students
- Delete a student by ID
- Data stored using SQLite database

## Technologies Used
- Python 3
- SQLite (sqlite3 module)

## How to Run
1. Open terminal in this folder
2. Run:
   python main.py

The database file will be created automatically.

## Learning Outcome
- Python–SQLite connection
- SQL CRUD operations
- Menu-driven Python programs
