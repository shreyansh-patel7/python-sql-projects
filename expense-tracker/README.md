# Expense Tracker (Python + SQLite)

This is a console-based Expense Tracker application built using Python and SQLite.
It helps users track daily expenses using a simple menu-driven interface.

## Features
- Add a new expense
- View all expenses
- Delete an expense by ID
- Persistent storage using SQLite

## Technologies Used
- Python 3
- SQLite (sqlite3 module)

## How to Run
1. Open terminal in this folder
2. Run:
   python main.py

Database file is created automatically.

## Learning Outcome
- Database schema design
- SQL INSERT, SELECT, DELETE
- Using AUTOINCREMENT primary keys
